# Mokhtari Architecture Website

A Django-based website developed for showcasing architectural projects.

## Features

- Home page
- Projects gallery
- About page
- Contact page
- Responsive design
- Optimized images (WebP)

## Technologies

- Python
- Django
- HTML5
- CSS3
- JavaScript

## Installation

```bash
git clone https://github.com/MahdieMokhtari/MokhtariProject
cd MokhtariProject
pip install -r requirements.txt
python manage.py runserver
```

Open:

http://127.0.0.1:8000/
